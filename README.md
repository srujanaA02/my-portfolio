# AKETI LAKSHMI SRUJANA — Developer Portfolio

A modern and fully responsive developer portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

The portfolio showcases my projects, technical skills, animations, parallax effects, and responsive UI design.

## 🚀 Live Portfolio

https://my-portfolio-chi-rust-49.vercel.app

## 📂 GitHub Repository

## 🛠️ Tech Stack

* React + Vite
* Tailwind CSS
* Framer Motion
* react-scroll-parallax
* JavaScript
* Responsive Design

## ✨ Features

* Fully responsive layout
* Animated hero section
* Parallax scrolling effects
* On-scroll animations
* Modern glassmorphism UI
* Mobile-friendly navigation
* Accessibility support using prefers-reduced-motion
